# מדריך קצר לעורך תוכן

כל התוכן המרכזי של דף הנחיתה נמצא בקובץ אחד:

```txt
src/content/site.json
```

## מה אפשר לעדכן בלי לגעת בקוד?

- כותרת ותיאור SEO
- שם העסק והסלוגן
- טלפון, מייל, כתובת ושעות פעילות
- קישורי Facebook / Instagram / TikTok
- טקסטים של אזור "למה לבחור בנו"
- מבצעים
- שאלות נפוצות
- טקסטים של אזור יצירת קשר

## עדכון מבצעים

פתח את:

```txt
src/content/site.json
```

מצא את החלק:

```json
"promotions": {
  "items": []
}
```

כל מבצע נראה כך:

```json
{
  "title": "מבצע בלמים",
  "subtitle": "רפידות וצלחות בלם במחיר מיוחד",
  "note": "מחיר / הטבה – קל לעדכון",
  "icon": "disc",
  "tone": "blue"
}
```

אפשר לשנות רק את הטקסטים. אם רוצים צבע כרטיס אחר, משנים:

```txt
blue / yellow
```

## אייקונים זמינים

```txt
battery, car, check, clock, disc, gauge, headphones, shield, spark, tag, target, truck, wrench
```

## בדיקת תקינות לאחר שינוי תוכן

אחרי שינוי בקובץ התוכן, להריץ:

```bash
npm run validate:content
```

אם יש שגיאה, ההודעה תראה בדיוק באיזה שדה יש בעיה.

## החלפת תמונת השלט

תמונת השלט נמצאת כאן:

```txt
public/images/sign.jpg
```

אפשר להחליף אותה בתמונה אחרת באותו שם.
